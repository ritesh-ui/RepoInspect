## 🔍 Detailed Attack Vectors

### 📍 Command Injection in `test_cases/vulnerable_data_flow.py`
- **Line**: 19
- **OWASP Category**: A01:2021-Broken Access Control
- **Severity**: Critical

> **Description**: The function constructs a shell command using user input without sanitization or validation, leading to potential command injection vulnerabilities.

#### 🏹 Attack Vector
An attacker could provide a malicious input such as '; rm -rf /; echo done' which would be executed by the shell, potentially compromising the system by deleting files or executing arbitrary commands.

#### 🛠 Remediation
Use the subprocess.run function without shell=True, and pass the command and arguments as a list to prevent shell injection. For example: subprocess.run(['echo', user_input])

### 📍 Command Injection in `test_cases/vulnerable_flow.js`
- **Line**: 11
- **OWASP Category**: A01:2021-Broken Access Control
- **Severity**: Critical

> **Description**: The function runUserCommand concatenates user input directly into a command that is executed by the exec function, leading to potential command injection vulnerabilities.

#### 🏹 Attack Vector
An attacker could provide input such as 'test; rm -rf /' which would be concatenated into the command, executing arbitrary commands on the server.

#### 🛠 Remediation
Use a library like 'child_process.execFile' which allows for passing user input as arguments, or sanitize the user input properly to ensure only whitelisted commands are executed.

- **Line**: 22

- **Severity**: High

> **Description**: The user input is being directly concatenated into a prompt for the AI model without any validation or sanitization, allowing an attacker to manipulate the prompt and control the output generated by the model.

#### 🏹 Attack Vector
An attacker could provide a malicious input that alters the prompt's meaning or introduces harmful instructions, leading to unintended or harmful model behavior. For example, `userInput` could be crafted to instruct the AI to share sensitive information or execute unexpected requests.
